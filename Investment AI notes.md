# Investment AI notes:

##### Project Roadmap (4 Weeks)

###### Week 1

- Set up React + FastAPI + MongoDB
- Build chat interface
- Search companies by ticker or name
- Fetch stock data from a free API

###### Week 2

- Implement financial ratio calculations
- Display financial statements and charts
- Add company comparison

###### Week 3

- Integrate an AI model for explanations and report generation
- Add news summarisation and watchlist

###### Week 4

- Polish the UI
- Add authentication
- Deploy the frontend and backend
- Write documentation and create a demo video
