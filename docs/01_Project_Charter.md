## What problem does it solve?

- Automatic financial calculations and finds the best case financial status of an firm.
- Company comparison
- Risk dedctor
- News Analysis

## Project scope

- Investment AI is a web application that helps users make informed investment decisions by providing stock and ETF analysis using financial data and AI-generated insights.
- The project includes user authentication, company analysis, financial metrics, portfolio tracking, and AI-powered investment recommendations.
- It is designed for educational purposes and long-term investment research.
- The project does not support live trading, cryptocurrency trading, or banking transactions.

## Success criteria

- The project will be considered successful if users can securely log in, analyze stocks and ETFs, receive AI-powered investment insights, track their portfolios, and access accurate financial information through a responsive and user-friendly web application
