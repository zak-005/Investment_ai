# Investment_ai

AI- powered investment research platform for the long-term company analysis

## What is this for?

An AI-powerd Web Application which helps the useres to analyze stocks and ETFs using financial data and AI insights.

## Why was it Build?

To make investing easier by providing company analysis, financial metrics, and AI-generated insights in one platform. It helps in time managment.

## Who is it for?

- Bignner investors
- Long-term investors
- Students learning about the Stock market

## Features

- [1] AI char
